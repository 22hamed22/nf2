NeRF Example — Docs

This repo contains a refactored NeRF example. See README.md for quick start instructions.

Contents

  • src/ — Python modules for data loading, model, training, and utils.
  • notebooks/ — Original notebook and (optionally) refactored notebook.

Usage

Open the Jupyter notebook from the repository root (so src is importable):

jupyter notebook notebooks/full_nerf_example_refactored.ipynb

API (high level)

Key exported symbols (also re-exported in src/__init__.py):

  • load_blender_data, get_ray_bundle
  • VeryTinyNeRFModel, ReplicateNeRFModel, positional_encoding
  • nerf_forward_pass, predict_and_render_radiance
  • volume_render_radiance_field, sample_pdf

For details, read the code in src/.
